# Tony Eneh Portfolio

A clean one-page developer portfolio for Tony Eneh, built with Next.js and Tailwind CSS.

Live: https://tony-eneh.vercel.app

## Sections

- Hero
- About
- Skills / Tech stack
- Experience timeline
- Projects
- Publications
- Education
- Contact

## Tech

- Next.js (App Router)
- React
- Tailwind CSS
- TypeScript

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
